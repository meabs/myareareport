# App Metadata Draft

## Name
MyAreaReport

## Short Description (max 80 chars)
UK area reports using public data: crime, flood, and planning.

## Long Description
MyAreaReport provides structured UK area reports using public datasets, including reported crime, flood warnings, and planning applications. It helps users understand local public information for a postcode or area. It does not provide emergency advice, property advice, crime prediction, or safety rankings.

## Category
Reference / Local Information

## Required Caveat Text
MyAreaReport uses public datasets that may be delayed, incomplete, or vary by provider. It should not be used for emergency decisions, legal decisions, insurance decisions, or personal safety decisions.

## Data Sources
- postcodes.io
- data.police.uk
- Environment Agency flood monitoring API
- planning.data.gov.uk
